
  # YouTube Thumbnail Template

  This is a code bundle for YouTube Thumbnail Template. The original project is available at https://www.figma.com/design/A1i4EejdpCPTBRwJyiH8GR/YouTube-Thumbnail-Template.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## GitHub Pages build

  Run `npm run build` to create the `dist` folder. The Vite config uses a
  relative base path in production so the site works on GitHub Pages under
  `https://<user>.github.io/<repo>/`. Publish the `dist` folder using your
  preferred GitHub Pages workflow (for example, deploy from a `gh-pages`
  branch or copy `dist` into `/docs`).
